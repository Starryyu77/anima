### [2026-01-23 11:10] 引入生成式形态引擎 (appearance_design.md)
- **目标 (Goal)**: 利用 LLM 的生成能力，让 Anima 的外观随用户投喂内容动态演化，增加“随机与确定”的平衡。
- **变更 (Changes)**:
  - 更新 `appearance_design.md`: 增加了“生成式形态引擎 (Generative Morphology Engine)”章节。
  - 定义了 **Data -> Genotype -> Phenotype** 的生成管线。
  - 确立了 **约束系统 (Constraint System)**，限制 LLM 只能生成抽象风格（几何/流体/光效），禁止具象化。
- **决策理由 (Reasoning)**:
  - 单纯的参数映射容易枯燥，LLM 作为“美术总监”可以创造出意想不到的形态，符合“生命体”不可预测的特性。
- **下一步 (Next)**:
  - 设计具体的 JSON 格式作为 LLM 生成的“基因数据包”。
